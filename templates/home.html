About our app

A cloud-based application to detect ChatGPT-generated scientific text and distinguish it from human-written text using machine learning techniques

The use of artificial intelligence is becoming increasingly prevalent in creating various types of data. However, there is a need to distinguish between data that has been generated by humans and data that has been generated by AI. Our proposed approach is to train a model using predefined datasets and deploy this model on a cloud-based service to predict the accuracy of text and determine whether it was created by a human or by AI.
